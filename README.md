# multipower_charge
这是一个关于电动公交充电优化的项目
每辆公交车有自己的运营计划，包含每个车次开始时间、结束时间、耗电量、耗电波动、耗时波动、空驶参数，对应车辆运营计划表。
耗电量是不确定的，但有一个范围；耗时也不确定，按照最大耗时即可；耗电量有鲁棒约束，约束值设为z
电池特性：充放电的电池损耗是不确定的，对应非线性损耗表；充电时间也是非线性的，对应非线性充电时间表。这两个特性也受充电功率的影响，分快充和慢充。
每个车次结束后都要面临选择：是否充电、去哪个充电站充电、选择哪个功率充电桩、充多少电、什么时候开始充电
目标函数：最小化总的充电电费+充放电损耗+空驶成本
约束条件：每个车辆分配一个方案，每个充电站的充电桩有限
主算法：列生成
子问题算法：动态规划，用来规划方案
